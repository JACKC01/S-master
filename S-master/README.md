# 今日头条后台管理系统

技术栈: vue2.x + elementUI

## 1.克隆项目

git clone git@gitee.com:smart-x/toutiao-pc.git

## 2.安装依赖

npm i || cnpm i

## 3.运行项目

npm run serve
